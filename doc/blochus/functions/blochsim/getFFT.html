getFFT

PURPOSE

calculates the FFT for a given time and signal pair

SYNOPSIS

function [Y,f] = getFFT(t,s)

DESCRIPTION 

getFFT calculates the FFT for a given time and signal pair

 Syntax:
       getFFT(t,m)

 Inputs:
       t - time [s]
       s - signal

 Outputs:
       Y - amplitudes of the FFT
       f - frequency [Hz]

 Example:
       getFFT(t,s)

 Other m-files required:
       none

 Subfunctions:
       none

 MAT-files required:
