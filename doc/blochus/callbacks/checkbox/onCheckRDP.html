onCheckRDP

PURPOSE

updates the checkboxes that activate relaxation during pulse

SYNOPSIS

function onCheckRDP(src,~)

DESCRIPTION 

onCheckRDP updates the checkboxes that activate relaxation during pulse
(RDP) or relaxation during switch-off (RDS)

 Syntax:
       onCheckRDP(src)
